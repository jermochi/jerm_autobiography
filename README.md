# Jerm's Autobiography (Streamlit)

This is a personal autobiography web application built with Streamlit for the Data Analytics and Visualizations Course

## What's Inside the App

- **Autobiography** - Learn more about me!
- **Portfolio** - Discover the things i've done!

## Setup Instructions

### Prerequisites
Make sure you have:
- Python 3.8 or higher installed on your computer
- pip (Python package manager) - usually comes with Python

### Step 1: Get the Code
First, clone or download this repository to your computer:
```bash
git clone <repository-url>
cd jerm_autobiography
```

### Step 2: Install Required Packages
Install the necessary Python packages using pip:
```bash
pip install -r requirements.txt
```

### Step 3: Run the App Locally
Start the Streamlit app by running:
```bash
streamlit run app.py
```
Your web browser should automatically open and show the app at `http://localhost:8501`. If it doesn't, just copy that URL into your browser!
